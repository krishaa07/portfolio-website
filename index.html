Who I am

I'm a final-year Computer Networking & IT Security student at Islington College, affiliated with London Metropolitan University. My coursework and internships have given me grounding in network protocols, web application security, and software quality assurance.

Through two internships — offensive security at YAJ Tech and QA at Nandi Solutions — I've worked on both sides of the software quality equation: finding vulnerabilities through penetration testing, and building systematic test coverage through structured QA processes.

I'm drawn to problems that sit at the intersection of security and quality — understanding how systems fail, why vulnerabilities exist, and what it takes to test them thoroughly. I'm early in my career and actively building on that foundation through projects and continuous learning.

Final Year Project · Full Stack

NetSecure — Network Topology Design & Security Validation Tool

A web-based tool for SMEs and non-expert IT users to design network topologies and validate security configurations. Includes IP conflict detection via BFS traversal, band-based security scoring, session persistence, and a security recommendations panel.

React.js Node.js Express.js MongoDB

Key learning: End-to-end system design for non-technical users; translating network security rules into actionable UI feedback.

Cybersecurity · Malware Analysis

Malware Analysis – LockBit 3.0 Ransomware Study

Analyzed LockBit 3.0 ransomware in a controlled lab environment. Examined file execution patterns, persistence mechanisms, encryption workflows, and indicators of compromise (IOCs). Findings compiled into a structured technical report.

VirtualBox Malware Analysis IOC Identification

Key learning: How ransomware establishes persistence and evades detection; communicating risk and impact clearly in written reports.

Web Security · Research

Researched SQL injection attack vectors and their real-world implications. Demonstrated exploitation techniques and proposed mitigation strategies, documented in a 3,000-word technical report.
